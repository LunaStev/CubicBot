# Cubic Discord Bot Plugin
This project is a bot that displays the current player count information of a Minecraft server in the Discord bot's status message.

# Features
* **Server Player Info Display**: The bot updates and shows the current player count and the maximum player count of the Minecraft server in real-time as the Discord bot's status message.
